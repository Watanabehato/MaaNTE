# MaaNTE에 오신 것을 환영합니다

**MaaNTE는 무료 오픈소스 프로젝트입니다. 어떤 형태로든 판매를 허가한 적이 없습니다.**

- Xianyu, Taobao 등의 플랫폼에서 본 소프트웨어를 구매하셨다면, 즉시 환불을 요청하고 판매자를 신고해 주십시오
- 이 팝업 스크린샷을 증거로 제출하여 환불을 요구하고 권리를 보호할 수 있습니다
- 리셀러에게 지불하는 모든 금액은 오픈소스 커뮤니티를 더욱 어렵게 만듭니다

> Mirror酱은 저희의 파트너로 다운로드 가속 서비스를 제공하며, 판매 행위에 해당하지 않습니다
> 현재 리소스 로딩은 간체 중국어를 가장 잘 지원합니다. 일부 텍스트 번역은 AI 번역에 의존합니다. 문제를 발견하시면 GitHub에 Issue를 제출해 주십시오.
