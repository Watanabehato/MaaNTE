# MaaNTE 问题排查手册

## 运行问题

### 无法启动

#### 弹窗提示To run this application, you must install .NET

- 前往[下载 .NET 10.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0) 并安装 `.NET 10.0 Desktop Runtime` （.NET 桌面运行时）。

### 无法连接窗口

1. 确保你打开了异环。
2. 将异环设为`1280x720`分辨率，窗口化。

## 任务问题

### 通用

#### 一开始就立刻完成

- 检查左侧`任务列表`中有没有勾选相应任务。
- 关闭游戏内插帧、超分辨率等影响画质的功能

#### 无法连接窗口/开始任务

- 确保你将 MaaNTE 添加到了杀毒软件的白名单里，或关闭了杀毒软件。
- 确保游戏语言为**简体中文**。

#### 无法正常进行点击/操作

- 确保你的 MaaNTE 处于**纯英文**路径下，并且没有全角符号。
- 确保你是**以管理员身份运行**的 MaaNTE 。
- 无法正常进行点击时尝试将鼠标输入方式改为`Seize`

#### 抢占鼠标问题

- 部分任务会强占鼠标，属于正常现象。

#### 识别失败

- 关闭游戏内插帧、超分辨率等影响画质的功能

### 自动钓鱼

- 见[任务问题-通用-无法连接窗口/开始任务](#无法连接窗口开始任务)

#### 不自动抛竿

- 见[任务问题-通用-无法正常进行点击/操作](#无法正常进行点击操作)

- 见[任务问题-通用-无法正常进行点击/操作](#无法正常进行点击操作)
